Quote generator Site Plan

Israel Lunceford

WDD 131.03

Overview

Purpose

To create a website that can inspire users by generating random inspirational quotes. I will allow the user to generate a random quote by the press of a button.

Audience

My audience for this website is probably my friends and family. However, It is really for anyone who would be interested.

Dynamic elements

I will include an event listener to generate a quote when a button is pressed. The event listener will incorporate an api which then generates a quote and will display through the use of another function. Another thing I want to do is add a switch that inverts the colors
